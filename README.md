# Tokens on Fire Dark

A cyberpunk neon dark theme for VS Code and Cursor, built on the **Tokens on Fire** brand system. Designed primarily for recording crisp, on-brand screencasts (Reels / shorts), but comfortable for everyday coding too.

## Palette

Pure-black editor with neon accents — the editor background matches the brand's content overlays, so there's no colour seam between screen recording and graphics.

| Role | Colour |
| --- | --- |
| Editor background | `#0D0D0D` |
| Body text | `#F5F5F5` |
| Cursor / keywords | `#FF006E` magenta |
| Functions / UI chrome / links | `#00F5FF` cyan |
| Strings | `#00FF85` neon green |
| Types / storage | `#FFB400` amber |
| Numbers | `#FF5400` orange |
| Comments | `#6A6A6A` grey (italic) |

## Install

**From VSIX:** Extensions panel → `…` menu → *Install from VSIX…* → pick `tokens-on-fire-vscode-theme-*.vsix`.

Then: `Cmd+K Cmd+T` → **Tokens on Fire Dark**.

## Recording tip

For screencasts, pair the theme with JetBrains Mono at a large size and a clean editor (hidden sidebar/minimap). A ready-made recording profile lives alongside the brand docs.
